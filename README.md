# PreisVERgleich

Tool zum Vergleichen der Preise der HardwareRat GmbH und verschiednenen Preisportalen wie z.B. Geizhals.

Es wird eine Differenz zwischen den beiden Preisen gebildet und dann visuell dargestellt.
